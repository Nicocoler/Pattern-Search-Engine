# PSE 特征工程系统设计

Version: 1.0  
Date: 2026-07-18

## 1. 模块定位

Feature Engine 是 PSE 的核心。它把行情和指标转成 AI、DTW、事件引擎都能使用的多维特征矩阵。

系统最终比较的不是 K 线图片，而是：

```text
window_size x feature_count
```

例如 60 日窗口与 40 个特征，形成 `60 x 40` 矩阵。

## 2. 特征分类

### 2.1 趋势特征

- `close_norm`：相对窗口首日收盘价归一化。
- `return_1d`、`return_5d`、`return_20d`。
- `ma5_above_ma10`。
- `ma10_above_ma20`。
- `ma20_slope`。
- `ma60_slope`。
- `ma_alignment_score`。

### 2.2 布林带特征

- `boll_upper_distance`：close 距离上轨百分比。
- `boll_middle_distance`：close 距离中轨百分比。
- `boll_lower_distance`：close 距离下轨百分比。
- `boll_width`。
- `boll_width_delta`。
- `boll_middle_slope`。
- `boll_upper_slope`。
- `is_near_upper`。
- `is_near_middle`。

### 2.3 成交量特征

- `volume_ratio_20`。
- `amount_ratio_20`。
- `pullback_volume_decay`。
- `breakout_volume_expand`。
- `volume_price_confirm`。

### 2.4 K 线结构特征

- `body_ratio`。
- `upper_shadow_ratio`。
- `lower_shadow_ratio`。
- `close_position_in_range`。
- `consecutive_up_days`。
- `consecutive_down_days`。
- `hammer_like`。
- `engulfing_like`。
- `stop_falling_signal`。

### 2.5 波动率特征

- `atr_ratio`。
- `high_low_range_ratio`。
- `volatility_20d`。
- `drawdown_from_window_high`。

## 3. 特征插件接口

每类特征实现为一个插件。

```python
class FeaturePlugin:
    name: str
    version: str
    required_columns: list[str]

    def calculate(self, bars: DataFrame, indicators: DataFrame) -> DataFrame:
        ...
```

推荐内置插件：

- `TrendFeaturePlugin`
- `BollFeaturePlugin`
- `VolumeFeaturePlugin`
- `CandleFeaturePlugin`
- `VolatilityFeaturePlugin`

## 4. 特征标准化

原则：

- 价格类特征尽量归一化为比例。
- 布林距离使用 `(close - band) / close`。
- 角度类特征使用斜率，不依赖图表像素。
- bool 特征存储为 0/1。
- 空值保持 null，并在 `quality_flags` 中说明。

## 5. 特征矩阵生成

推荐接口：

```python
def build_feature_matrix(
    symbol: str,
    end_date: date,
    window_size: int,
    feature_version: str,
) -> FeatureMatrix:
    ...
```

输出：

- `matrix`：二维数组。
- `feature_names`：特征顺序。
- `quality`：窗口质量。

## 6. 特征版本

任何新增、删除、改名、改公式都必须升级 `feature_version`。

示例：

```text
feat_v1.0.0:
  trend_v1
  boll_v1
  volume_v1
  candle_v1
  volatility_v1
```

## 7. 质量标记

`quality_flags` 示例：

```json
{
  "missing_indicator": false,
  "window_length": 60,
  "suspended_days": 2,
  "has_extreme_gap": false
}
```

## 8. 验收标准

- 同一输入在同一版本下输出完全一致。
- 特征顺序稳定。
- 单只股票最近 60 日特征矩阵可在 50ms 内生成。
- 全市场每日增量特征生成可批量执行。
