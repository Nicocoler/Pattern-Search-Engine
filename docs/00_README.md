# Pattern Search Engine（PSE）开发手册

本目录保存“基于 K 线形态相似搜索的智能选股系统”的本地开发文档。PSE 的目标不是预测涨跌，也不是传统条件选股器，而是：

> 输入一个用户认可的历史走势模板，在全市场股票中搜索当前走势最相似的股票，并给出相似度、阶段判断、相似原因和风险提示。

## 核心思想

系统不直接比较 K 线图片，也不把所有规则写死在 DTW 里。

推荐架构是：

```text
市场数据 -> 数据中心清洗(AkShare) -> 指标计算 -> 多维特征矩阵 -> 高斯模糊事件识别 -> 时间弹性形态匹配(加权DTW) -> AI 自适应排序 -> 可解释结果与回测验证
```

核心算法组合：

- **行情源**：完整适配开源行情库 AkShare。
- **多维特征**：趋势、布林带（主创涨跌幅限制归一化系数 LNF 消除板块波幅干扰）、成交量、K 线结构、波动率。
- **高斯模糊事件链**：上涨、触上轨、回调、回踩中轨、中轨支撑、缩量、放量启动。采用单峰高斯置信度函数（$0 \sim 1.0$）替代硬阈值。
- **高性能 DTW**：加入 Sakoe-Chiba 限宽约束（带宽 $W$ 限制对角线搜索），将复杂度优化到 $O(N)$，解决不同股票节奏快慢不同的问题。
- **自适应加权评分**：引入基于好/坏反馈的 L1 归一化自适应权重微调算法，让系统越用越懂朔哥哥的看盘感。
- **历史无偏回测**：加入第 16 章形态回测系统，自动追踪多周期胜率与 Alpha 表现。

## 推荐第一阶段

先实现单机 MVP：

1. 获取 A 股日 K 数据。
2. 计算 MA、BOLL、成交量倍率、ATR、K 线结构。
3. 内置“布林上轨 -> 回踩中轨 -> 二次启动”模板。
4. 对全市场最近 60 日窗口做相似度扫描。
5. 输出 Top N 股票、评分拆解、可解释原因和风险提示。

## 风险声明

本系统用于形态搜索与研究辅助，不构成投资建议，不直接输出买卖指令。
