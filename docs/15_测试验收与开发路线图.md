# PSE 测试验收与开发路线图

Version: 1.1  
Date: 2026-07-18

## 1. 测试目标

PSE 的测试重点不仅是看接口通不通，而是要验证形态搜索和回测系统在复杂的 A 股特殊制度和除权跳空环境下，结果是否**稳定、无未来函数、自适应对齐且100%可复现**。

## 2. 单元测试

覆盖：
- 指标计算：MA、BOLL、ATR、成交量倍率在停牌、数据不足和零成交量下的极值与 null 填充测试。
- 特征插件：主创涨跌幅限制归一化系数 (LNF)、一字板特征插值有效性测试。
- 事件检测：高斯置信度函数输出值的平滑衰减度检验。
- 相似度：Sakoe-Chiba 带宽限宽对角线约束下的 DTW 时间对齐对错及速度。
- 排序反馈：正负反馈提交后，L1 归一化特征权重微调梯度公式方向正确性。

## 3. 集成测试（防止未来函数审计）

回测和形态匹配必须通过严密的 **未来函数自动化隔离审计 (Look-ahead bias containment test)**：
- 构造一个 60 日历史样本 $t_0 \sim t$，故意在 $t+1$ 日构造一个超额暴涨信号。
- 传入截至 $t$ 日的数据给系统，确保计算出来的相似得分与最终推荐列表，绝不受到 $t+1$ 日暴涨数据的任何侧向影响（结果差异为 0），验证隔离成功。

## 4. 回归测试样本集

建立固定测试样本集：
```text
tests/fixtures/
  stocks.csv                     # 包含主板、创业板、st个股的样本
  daily_bars_sample.csv          # 包含除权、停牌、一字板的日K行情样本
  template_boll_pullback.json    # 布林回踩经典抽象模板配置
  expected_top_results.json      # 已知高相似度的期望输出及分项评分明细
```
每次指标、特征插件、高斯置信度参数或者对齐算法修改后，必须对样本集重跑相似度扫描，验证 Top 10 排序稳定和分项相似度解释描述逻辑无偏。

## 5. 性能与容量测试

* **单机并发同步限制**：验证 Data Center 在 5 个并发协程和随机延迟下，AkShare 接口能稳定获取 5300 股日 K 且不被风控封锁。
* **匹配吞吐量**：全市场 5300 股最近 60 日单窗口扫描（经过两级预过滤，限宽 DTW 计算）在单机下耗时小于 3 分钟。
* **回测历时性能**：单只形态模板执行 3 年历史日滚动回测（包含防未来函数指标切叠），总耗时控制在 10 分钟以内。

## 6. 开发路线图 (Roadmap)

### Phase 0 文档与基础设施
- 完成本文档及 16 份详细设计方案。
- 初始化 PostgreSQL/TimescaleDB 表结构（stocks 板块、st 字段及 `dirty_factors` 任务表）。

### Phase 1 行情清洗与数据质量
- 数据中心对接 AkShare 获取全市场 spot 及前复权 hist 日 K。
- 实现晚间 18:30 “哨兵保护网关”错峰更新机制。
- 编写个股发生分红除权后的“`dirty_factors` 差分核算与全历史重算触发器”。

### Phase 2 特征插件与高斯事件检测
- 编写 LNF 板块涨跌幅限制特征归一化插件及“一字板”插值修正程序。
- 实现 TREND_UP 等 8 个 MVP 事件的高斯模糊连续置信度判定函数（输出平滑的 $0 \sim 1.0$ 置信度）。

### Phase 3 多维 DTW 相似度与自适应反馈
- 引入局部 Z-score 标准化模块，消除量纲冲突。
- 引入基于 C / Numba 加速的 Sakoe-Chiba 限宽约束（$W = floor(0.15 \times L)$）多维加权对齐。
- 实现规则加权在线自适应 L1 微调（反馈好/坏自动调整模板维度权重）。

### Phase 4 API 与前端可视化
- 模板/扫描/对比 API 设计。
- 前端 `KlineCompareChart` 支持基准百分比归一化同屏重叠对比及 K 线事件气泡标注（Event Markers）。

### Phase 5 形态回测系统开发
- 实现历史交易日滑动仿真交易（Sliding Simulation）及持股 5d, 10d, 20d 绩效汇总。
- 编写防未来函数切片审计组件。
- 前端实现形态回测累计净值曲线对比沪深 300 业绩看板。

### Phase 6 AI 与多模板扩展
- 当用户标注累积足够后，引入 LightGBM Ranker 训练。
- 多模板自动发现与推荐。

## 7. 最终交付与验收清单

MVP 第一版交付必须无死角满足：
- [x] **数据源合规**：稳定通过 AkShare 网络并发拉取股票列表与前复权日 K，带有自适应随机休眠。
- [x] **数据高品质**：个股除权能由差分器捕捉，自动全量重算指标与特征历史，无断层。
- [x] **匹配抗噪性**：事件识别引擎采用高斯置信度，边缘回踩及量能变动不被一刀切错失。
- [x] **算法高效率**：两级预过滤及限宽 DTW 计算在 3 分钟内输出全市场推荐。
- [x] **可解释对齐**：前端详情页支持起始日归零的百分比拟合 K 线及事件气泡标注交互。
- [x] **自学习特性**：提交正负反馈后，形态模板权重能够自动微调更新，L1 归和始终为 1.0。
- [x] **历史胜率检验**：支持形态回测，无偏统计历史 5d, 10d, 20d 形态表现，输出净值曲线。
